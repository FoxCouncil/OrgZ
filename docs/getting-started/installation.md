# Installation

## Windows

OrgZ is distributed as a self-updating Windows application via [Velopack](https://velopack.io/).

1. Download the latest release from [GitHub Releases](https://github.com/FoxCouncil/OrgZ/releases)
2. Run the installer
3. OrgZ will automatically check for updates on launch

## Requirements

- Windows 10 or later
- .NET 10 runtime (bundled with the installer)
